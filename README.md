# Homelessness in the Pacific Northwest

This project explores homelessness as both a social and human issue, focusing on its impact in the Pacific Northwest.

## 🔗 Live Website
https://ackerjon8.github.io/homelessness-capstone/

## 📊 Project Components
- Infographic (data + statistics)
- Video Remix (human perspective)
- Works Cited (MLA sources)

## 🎯 Purpose
The goal of this project is to raise awareness and present both the causes and potential solutions to homelessness.
